## 💡 About Me
새로운 기술을 빠르게 익히고 실제 문제에 적용하는 과정에 몰입하는 **응용 소프트웨어 개발자**입니다. 
단순한 기능 구현을 넘어 **사용자 중심의 UI/UX 개선**과 **다양한 환경을 고려한 시스템 설계**에 관심이 많습니다. 혼자보다는 팀 단위의 협업을 통해 문제를 다각도로 바라보고 해결하는 것을 즐깁니다.

- 🔭 현재 프론트엔드(React, TypeScript) 역량을 기반으로 AI와 IoT 등 다양한 도메인을 넘나들며, 사용자에게 최적의 디지털 경험을 제공하기 위해 시야를 넓히고 있습니다.
- 🌱 KCI 등재 논문 연구 및 KISIA 보안 크루 활동을 통해, 시스템의 깊은 구조와 안전성까지 고려하는 탄탄한 개발 기본기를 다졌습니다.
- 🎯 작은 변화라도 구조를 설계하고 실행으로 옮기는 것을 중요하게 생각합니다.
